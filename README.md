## Project Overview

This repository contains project materials for a daily diary study
examining alcohol involvement, sexual event context, and distal
individual factors in relation to verbal and nonverbal sexual consent
cue use.

## Repository Structure

Primary materials in this project repo include

-   `index.qmd`: the manuscript (dissertation write-up)

Quarto notebooks saved in the /notebooks folder

-    `01_clean_cues`: data cleaning, screening, preprocessing

-   `02_imp_cues`: missingness and multiple imputation

-   `03_prep_cues`: variable construction and model preparation

-   `04_ana_cues`: analyses for hypothesis testing
